# Image Assets

Place your images in the following folders:

## `/public/images/`

- `profile-placeholder.jpg` - Your professional photo (replace this placeholder)

## `/public/images/logos/`

Company logos for experience section:

- `barclays.png` - Barclays logo
- `persistent.png` - Persistent Systems logo
- `freelance.png` - Freelance/general icon

## `/public/images/projects/`

Project screenshots/thumbnails:

- `shrinkr.jpg` - Shrinkr project screenshot
- `memecasso.jpg` - MemeCasso project screenshot
- `paathshala.jpg` - Paathshala project screenshot
- `devsync.jpg` - DevSync project screenshot
- `anonaadhar.jpg` - AnonAadhaar project screenshot
- `taskflow.jpg` - TaskFlow project screenshot

## `/public/images/tech/`

Custom tech icons (if CDN fails):

- `ethereum.svg`
- `web3.svg`
- `hardhat.svg`

## Note

Most skill icons are loaded from CDN (cdn.jsdelivr.net/gh/devicons).
If you want to use local images, replace the URLs in `data/skills.json`
